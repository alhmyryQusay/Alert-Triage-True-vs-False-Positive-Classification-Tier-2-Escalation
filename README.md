# Alert Triage, True vs False Positive Classification & Tier 2 Escalation

## Project Description
In this project, I worked within a **Security Operations Center (SOC)** SIEM environment. I was responsible for analyzing 35 security alerts generated by monitoring systems.  
The main objective was to triage each alert, determine whether it was a **True Positive** or **False Positive**, take appropriate actions, and escalate confirmed incidents to **SOC Tier 2**.

## Key Features
- Analyze security alerts in a SOC environment.
- Classify alerts as True Positive or False Positive.
- Take appropriate actions for each alert.
- Escalate confirmed incidents to Tier 2.

## Tools & Technologies Used
- SIEM (Security Information and Event Management
- Splunk
- Security monitoring and alerting tools
- SOC incident escalation procedures

## Usage
This project showcases my experience in handling security alerts. It demonstrates the process of alert triage, classification, and escalation in a professional SOC environment.
